== DESCRIPTION ==
Calendarize It! - a powerful Calendar and Event plugin for WordPress. With this Free Add-on you can add additional jQuery UI Themes to Calendarize It! 

== INSTALLATION ==

1. Upload the 'calendarize-it-ui-themes' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. In the menu you will find Calendarize It. In the Options Panel you can choose between the different UI themes.
